## Project Title

Cloud-Based Car Maintenance Reminder System

---

## Project Description

This project is a simple cloud-hosted website designed to remind users about important car maintenance tasks such as oil changes, tyre pressure checks, battery inspections, and vehicle servicing.

The website is hosted on an AWS EC2 virtual server using Nginx and secured using HTTPS/TLS.

---

## Features

- Oil change reminders
- Tyre pressure checks
- Battery inspection reminders
- Vehicle servicing reminders
- HTTPS secure website
- Cloud hosting using AWS EC2
- Linux shell backup script
